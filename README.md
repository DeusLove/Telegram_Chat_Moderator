# Telegram Moderator Bot

Бот для автоматической модерации чатов в Telegram, использующий дообученные модели на базе `deepvk/RuModernBERT-base` для детекции спама и токсичного контента.

## Описание

Данный бот предназначен для автоматической модерации групповых чатов. Он анализирует все текстовые сообщения участников и применяет меры воздействия при обнаружении спама или токсичного контента. Бот использует две дообученные модели:
- **Модель токсичности** — определяет уровень агрессии и оскорблений
- **Модель спама** — выявляет рекламные и нежелательные сообщения

## Функционал

### Основные возможности
- Автоматическая проверка всех текстовых сообщений
- Удаление запрещённого контента
- Система прогрессивных мутов (от 1 минуты до 7 дней)
- Блокировка нарушителей за спам или систематические нарушения
- История нарушений с автоматическим сбросом через 30 дней
- Логирование всех действий

### Система наказаний

| Нарушение | Мера воздействия |
|-----------|------------------|
| Спам | Удаление + немедленная блокировка |
| Токсичность (1) | Мут 1 минута |
| Токсичность (2) | Мут 5 минут |
| Токсичность (3) | Мут 15 минут |
| Токсичность (4) | Мут 1 час |
| Токсичность (5) | Мут 6 часов |
| Токсичность (6) | Мут 24 часа |
| Токсичность (7) | Мут 3 дня |
| Токсичность (8) | Мут 7 дней |
| Токсичность (>8) | Блокировка |
